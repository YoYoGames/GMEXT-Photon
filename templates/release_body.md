## DESCRIPTION

The Photon extension brings Exit Games' Photon Cloud to GameMaker, providing real-time multiplayer (rooms, matchmaking and events), text Chat and Voice — so players can find each other, share game state and talk in real time across Windows, macOS, Linux, Android and iOS.

## FEATURES

- **Realtime**: connect to the Photon Cloud, join lobbies, create/join/leave rooms, matchmaking, and send/receive custom string and binary events.
- **Room & Player Properties**: shared key/value state synced across the room, with compare-and-swap updates.
- **Chat**: channel subscriptions, public and private messaging, presence and friend status.
- **Voice**: real-time audio streams between players, with per-player mute and volume control.
